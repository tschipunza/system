About Roles & Permissions
  • System Roles are predefined (employee, manager, director, admin)
  • Custom Roles can be created for specific needs
  • Each role can have multiple permissions assigned
  • Employees are assigned one role that determines their access
  • System roles cannot be deleted, only modified
Default Role Hierarchy
  1. Admin - Full system access (all permissions)
  2. Director - Advanced access (approvals, reports)
  3. Manager - Team management (assignments, reviews)
  4. Employee - Basic access (own records only)
To assign roles to employees, go to Settings → Employees
