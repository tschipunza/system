About Database Backups
  • Create Backup: Creates a complete snapshot of your database
  • Download: Download backup file to your computer
  • Restore: Replace current database with backup data

Best Practices
  • Create backups regularly (daily or weekly recommended)
  • Download important backups to external storage
  • Test restore process periodically
  • Keep at least 3-5 recent backups
  • Create backup before major system changes
